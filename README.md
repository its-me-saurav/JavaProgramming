# JavaProgramming
“Java在线测试平台”是一个面向Java语言学习者综合性的在线测试学习平台，其拥有丰富的测试题库、PPT学习资料等。
它的错题本、收藏夹、PPT下载等功能突破了传统答题模式的限制，将时尚的答题模式应用到课内教育上。

Web站：http://java.zjut.edu.cn
